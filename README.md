# Bootstrap Custom Template Project

A Bootstrap-based landing page project featuring responsive layout and UI components.

## 🚀 Overview
This project is a responsive landing page built using **Bootstrap 5** and **custom CSS**.  
It demonstrates how Bootstrap components and grid system can be used to create a modern and clean website layout.

The template includes multiple sections commonly used in product landing pages.

## 🛠 Technologies Used
- HTML5
- CSS3
- Bootstrap 5

## 📌 Features
- Responsive navigation bar
- Hero / Home section
- About section with cards
- Features section
- Results statistics section
- Customer reviews carousel
- Pricing cards
- FAQ accordion
- Email signup section
- Footer with social icons

## 📂 Project Structure
```
bootstrap-custom-template-project
│
├── index.html
├── style.css
├── images/
│   └── pictures/
```

## 📱 Responsive Design
The layout is fully responsive and adapts to different screen sizes using **Bootstrap grid system**.

This project was created as part of my **front-end development learning journey** to practice Bootstrap layout, components, and responsive design.
